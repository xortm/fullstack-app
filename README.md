# Full Stack Application

完整的全栈应用，整合前端和后端。

## 🏗️ 项目结构

```
fullstack-app/
├── frontend/          # 前端（React）
│   ├── src/
│   ├── public/
│   ├── package.json
│   └── vite.config.js
└── backend/           # 后端（Express）
    ├── routes/
    ├── config/
    ├── server.js
    └── package.json
```

## 🚀 快速开始

### 后端

```bash
cd backend
npm install
npm start
```

服务器将在 http://localhost:3000 启动

### 前端

```bash
cd frontend
npm install
npm run dev
```

应用将在 http://localhost:5173 启动

## 📦 功能特性

- ✅ 用户注册和登录
- ✅ JWT 认证
- ✅ 文章 CRUD
- ✅ 响应式设计
- ✅ 自动部署

## 🛠️ 技术栈

### 前端
- React 18
- Vite
- React Router
- Tailwind CSS

### 后端
- Node.js
- Express
- SQLite
- JWT
- bcrypt
